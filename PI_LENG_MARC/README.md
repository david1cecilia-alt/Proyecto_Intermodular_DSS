El XML desarrollado representa la información de los alumnos de la autoescuela, incluyendo sus datos personales, su estado dentro del sistema y su historial de clases prácticas, reflejando una parte coherente de la base de datos del sistema. La estructura está organizada de forma jerárquica, donde la autoescuela contiene alumnos, y cada alumno incluye sus datos personales junto con la información relacionada con su actividad. Esto permite simular una exportación de datos desde la base de datos a un formato estructurado y fácilmente intercambiable.

Para asegurar la validez de los datos, se ha diseñado un esquema XSD que define la estructura del XML, los tipos de datos y diferentes restricciones, como enumeraciones para el estado del alumno, patrones para validar el formato del DNI, restricciones sobre tipos de datos como fechas en formato dateTime y enteros para identificadores y duraciones, así como la obligatoriedad de ciertos atributos como los identificadores de alumno y clase.

La validación de los documentos se ha realizado mediante un script desarrollado en Node.js utilizando la librería libxmljs2, que permite leer, parsear y validar los archivos XML contra el esquema XSD de forma automática. Este proceso muestra por consola si el XML es válido o no, y en caso de error, detalla los fallos encontrados indicando el mensaje y la línea correspondiente, lo que facilita su detección y corrección.

La validación se ha comprobado mediante el uso de un XML correcto y otro con errores intencionados. En el caso del XML erróneo, el sistema detecta fallos como valores no permitidos en enumeraciones, formatos incorrectos en el DNI, tipos de datos inválidos o errores en campos como la fecha y la duración, demostrando que tanto el esquema como el sistema de validación funcionan correctamente.

Este proyecto integra XML, XSD y Node.js como un sistema completo de gestión y validación de datos, permitiendo simular un entorno real de control de información, útil para la interoperabilidad, la integridad de los datos y posibles integraciones con otros sistemas.
